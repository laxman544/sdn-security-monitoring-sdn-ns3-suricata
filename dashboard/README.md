# SDN Security Monitoring – Dashboard

This folder contains the Python **Plotly Dash** application used to visualize
security events detected in the *Security Network Monitoring in SDN* project.

The dashboard reads:

- **Suricata alerts** from `../suricata/eve.json`
- (Optionally) **NS-3 flow statistics** exported to CSV (e.g.
  `../ns3/ns3_flow_stats.csv`)

It then displays:

- Top attacker IPs by alert/packet count  
- Alerts over time  
- Distribution of attack categories
